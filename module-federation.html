Angular (V15) Module Federation Training

1. Introduction to Micro Frontends and Module Federation

Start Time: 10:00 AM | End Time: 10:15 AM

Micro Frontends allow you to split large front-end applications into smaller, manageable parts. Module Federation enables sharing modules across applications.

2. Setup and Configuration for Module Federation

Start Time: 10:20 AM | End Time: 10:35 AM

In this segment, we’ll install the necessary packages and configure Webpack for Module Federation.

// Installing Webpack 5
npm install --save @angular-architects/module-federation

// Sample webpack.config.js
module.exports = {
    name: "app",
    remotes: {
        "remoteApp": "remoteApp@http://localhost:3001/remoteEntry.js",
    },
    exposes: {
        "./Component": "./src/app/component.ts",
    },
    shared: {
        "@angular/core": { singleton: true },
        "@angular/common": { singleton: true },
    }
};
